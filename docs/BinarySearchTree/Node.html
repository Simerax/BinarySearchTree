    class BinarySearchTree::Node(T)

    • BinarySearchTree::Node(T)
    • Reference
    • Object

    Overview

    The Node Class is used to create Binary Trees. Every Node holds data. Every Node can have a left and a right child. Non existing Childs are nil

    root = BinarySearchTree::Node(Int32).new(15)
root.insert(5)        # will be the left child of root
root.insert(20)       # will be the right child of root
root.insert(18)       # will be the left child of '20'
root.find(20)         # => BinarySearchTree::Node(Int32)
root.find(256)        # => nil
root.delete_node(18)  # => true
root.delete_node(256) # => false

    def insert(val : T) #

    Inserts a new element into the tree

    Insertion is done by finding a Node(T) which does not have a child yet and appending it to that one (by creating a new Node(T)) The Child is being inserted in the left child if the data is smaller then the one of the current Node(T) otherwise in the right one

    root = BinarySearchTree::Node(Int32).new(15)
root.insert(5)
